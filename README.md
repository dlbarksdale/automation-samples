test automation, API validation, and CI integration.

Most of my professional work has been performed on proprietary systems and
private repositories. These samples are intentionally minimal and anonymized,
and are meant to demonstrate patterns, structure, and decision-making rather
than full production implementations.

- API testing patterns using Pytest
- Clear, maintainable test structure
- CI integration concepts (GitHub Actions)
- A pragmatic, reliability-focused approach to quality engineering
